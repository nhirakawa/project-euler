## How to run
```
mvn package
java -jar ./target/project-euler-1.0-SNAPSHOT.jar -{problem name}
```

### Example
```
java -jar ./target/project-euler-1.0-SNAPSHOT.jar -problem3
Solved Problem3 in 37 ms [6857]
```
